# 🧠 ML Pipeline Dashboard using Streamlit

An interactive and visually appealing **Machine Learning Pipeline Dashboard** built with Streamlit that enables users to perform end-to-end ML workflows without writing code.

---

## 🚀 Features

### 1️⃣ Problem Selection
- Supports:
  - Classification
  - Regression

---

### 2️⃣ Data Input & Visualization
- Upload datasets (CSV / Excel)
- Select target feature
- Choose input features
- PCA-based visualization for data understanding

---

### 3️⃣ Exploratory Data Analysis (EDA)
- Statistical summary of dataset
- Missing values visualization
- Distribution plots
- Correlation heatmap
- Target variable analysis

---

### 4️⃣ Data Cleaning & Engineering
- Handle missing values:
  - Mean
  - Median
  - Mode
- Outlier detection:
  - IQR
  - Isolation Forest
  - DBSCAN
  - OPTICS
- Option to remove outliers interactively

---

### 5️⃣ Feature Selection
- Variance Threshold
- Correlation with target
- Information Gain

---

### 6️⃣ Data Splitting
- Train-test split
- Customizable test size
- Handles both regression and classification correctly

---

### 7️⃣ Model Selection
- Linear Regression
- Logistic Regression
- SVM (with kernel options)
- Random Forest
- KMeans (unsupervised learning)

---

### 8️⃣ Model Training & Validation
- K-Fold Cross Validation
- Model performance tracking

---

### 9️⃣ Performance Metrics
#### Classification:
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

#### Regression:
- R² Score
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)

✔ Overfitting and Underfitting Detection

---

### 🔟 Hyperparameter Tuning
- Grid Search
- Random Search
- Compare performance before and after tuning

---
